dup(Int32)

Duplicates the value at index i relative to the top of the stack and pushes it as copy on top of the stack.

Declaration
public void dup(int i = 0)
Parameters
Type Name Description
System.Int32 i

Relative index of a stack entry.
Remarks

Since dup-operations always require memory copies, it is more efficient to use swap when possible. see: mark() for absolute indices.

dup(UInt32)

Duplicates the value at index i as absolute index in the stack and pushes it as copy on top of the stack.

Declaration
public void dup(uint i)
Parameters
Type Name Description
System.UInt32 i

Absolute index of a stack entry.
Remarks

Since dup-operations always require memory copies, it is more efficient to use swp() when possible. see: mark() for absolute indices.
