Method cmpi

cmpi(Int32, Int32)

Compares the values at index a relative to the top of the stack with the System.Int32 value b.

Declaration
public int cmpi(int a, int b)
Parameters
Type Name Description
System.Int32 a

Relative index of the first stack entry.
System.Int32 b

An System.Int32 value to compare to.
Returns
Type Description
System.Int32

-1 – value at a is less than value b.
0 – value at a is equal to value b.
1 – value at a is greater than value b.
