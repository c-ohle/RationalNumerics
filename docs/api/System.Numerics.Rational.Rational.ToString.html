ToString()

Converts the numeric value of the current Rational instance to its equivalent string representation.

Declaration
public override readonly string ToString()
Returns
Type Description
System.String

The string representation of the current Rational value.
Overrides
System.ValueType.ToString()
Remarks

Behavior like System.Double.ToString with decimal notation in the range E-04 to E+16
Special: Maximum number of digits is by default 32.
Special: Repetions marked with apostrophe (') like 1/3 as "0.'3" instead of "0.333333…"
Special: Overflow marked with trailing dots (…) like: "1.234567…"

ToString(String, IFormatProvider)

Formats this Rational instance to its string representation by using the specified format and culture-specific format information.

Declaration
public readonly string ToString(string format, IFormatProvider provider = null)
Parameters
Type Name Description
System.String format

A standard or custom numeric format string.
System.IFormatProvider provider

An object that supplies culture-specific formatting information.
Returns
Type Description
System.String

The string representation of the current Rational value as specified by the format and provider parameters.
Implements
System.IFormattable.ToString(System.String, System.IFormatProvider)
Remarks

Supported numeric standard formats: C, D, E, F, G, N, P, R like "E10".
Custom formats supported, currently in the range +/- E-308 .. E+308 like "0.#####"
Standard formats F, E with unlimited many digits like: "F1000" for 1000 decimal digits.
Special R: behavior like System.Double.ToString(System.String) with fallback to format Q to ensure the round-trip to an identical number.
Additional: Format Q for Fraction notation returns "1/3" instead of "0.333333…"
Additional: Format L for Large, like F without trailing zeros: L5 returns 1.5 instead of 1.50000…
